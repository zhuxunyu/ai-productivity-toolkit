## 🚫 绝对禁止内容

### 1. 个人信息（严禁暴露）
```
❌ 真实姓名
❌ 手机号码
❌ 个人邮箱
❌ 微信/QQ 号（用企业号）
❌ 住址/定位
❌ 身份证照片
❌ 护照/驾驶证
❌ 学生证/工牌
❌ 真实照片（用人设图）
```

### 2. 技术凭证（严禁暴露）
```
❌ API Key
❌ Access Token
❌ 密码
❌ 私钥文件
❌ 数据库连接信息
❌ 服务器 IP
❌ 后台截图（打码也不行）
```

### 3. 财务信息（严禁暴露）
```
❌ 银行卡号
❌ 支付宝/微信收款码（用企业号）
❌ 收入截图（可模糊处理）
❌ 纳税记录
```

### 4. 公司信息（严禁暴露）
```
❌ 公司名称
❌ 办公地点
❌ 同事信息
❌ 内部文档
❌ 工作群聊截图
```

## ✅ 安全操作指南

### 账号注册
```
✅ 使用专用手机号（非个人主号）
✅ 使用专用邮箱（如 protonmail）
✅ 企业号认证（用个体户执照）
✅ 收款用企业账户（非个人）
```

### 内容创作
```
✅ 用虚拟人设（不透露真实身份）
✅ 用 AI 生成头像（不用真实照片）
✅ 内容聚焦工具/教程（不聊个人）
✅ 案例用公开数据（不用内部数据）
```

### 引流变现
```
✅ 用企业微信（非个人微信）
✅ 用知识店铺（平台官方）
✅ 用平台电商（小红书店铺）
✅ 合同用公司名义（非个人）
```

### 代码/工具分享
```
✅ GitHub 用组织账号（非个人）
✅ API Key 用环境变量（不硬编码）
✅ 示例代码脱敏（替换关键信息）
✅ 截图打码（敏感信息模糊）
```

## ⚠️ 风险警示

### 高风险行为（绝对禁止）
1. **个人微信引流** - 可能被封号/骚扰
2. **暴露真实身份** - 隐私泄露风险
3. **分享 API Key** - 被盗用/盗刷
4. **晒收入截图** - 被举报/被借
5. **透露公司信息** - 可能违反竞业

### 中风险行为（谨慎操作）
1. **语音/视频出镜** - 用人设/动画代替
2. **直播互动** - 用虚拟形象/不露脸
3. **线下活动** - 用公司名义/公共场合
4. **合作签约** - 用公司合同/律师审核

### 低风险行为（可以操作）
1. **图文笔记** - 主流形式
2. **AI 配音** - 不用真实声音
3. **平台电商** - 官方保障
4. **知识付费** - 正规渠道

## 🔐 技术安全措施

### 代码脱敏
```python
# ❌ 错误示例（暴露 API Key）
API_KEY = "sk-1234567890abcdef"

# ✅ 正确示例（环境变量）
import os
API_KEY = os.getenv("API_KEY")  # 不提交到 Git
```

### 截图处理
```
✅ 用马赛克模糊敏感信息
✅ 用贴纸覆盖关键数据
✅ 用示例数据替换真实数据
✅ 用示意图代替真实截图
```

### Git 安全
```bash
# .gitignore 必须包含
.env
*.key
*.pem
secrets/
config.local.py
```

## 📞 应急处理

### 发现隐私泄露
1. **立即删除**相关笔记
2. **修改密码**相关账号
3. **联系客服**举报侵权
4. **保留证据**必要时报警

### 被人肉/骚扰
1. **开启隐私保护**（平台设置）
2. **拉黑举报**恶意用户
3. **固定证据**截图保存
4. **报警处理**严重情况

### 账号异常
1. **立即下线**所有设备
2. **修改密码**+ 二次验证
3. **联系平台**申诉解封
4. **准备备用**账号方案
