### 回答 1：AI 效率工具推荐

```markdown
作为 AI 研究员，我日常需要处理大量数据和文档，试过很多工具，最近把自己常用的功能整理成了一个开源工具库。

分享几个我觉得最实用的：

## 1. ChatGPT/Claude + Prompt 优化

大部分人用 AI 效果不好，是因为 prompt 写得不够好。

举个例子：

❌ 糟糕的 prompt：
"帮我写一个 Python 脚本"

✅ 优化后的 prompt：
"你是一位资深 Python 开发者。请编写代码：
【代码要求】
1. 功能完整，逻辑清晰
2. 遵循最佳实践和代码规范
3. 包含必要的注释和文档字符串
4. 考虑边界情况和错误处理
5. 提供使用示例

【功能需求】
[具体需求]"

效果天差地别。

## 2. Excel 自动化工具

如果要批量处理 Excel 文件，手动操作太慢了。

我常用的是：
- openpyxl - Python 库，适合编程
- Power Query - Excel 自带，无需编程
- 我自己写的工具库 - 一键批量处理

## 3. 数据爬取框架

需要采集公开数据时，我用 Python + requests + BeautifulSoup。

注意几点：
- 只爬公开数据
- 遵守 robots.txt
- 控制请求频率（建议 2 秒/次）
- 不突破反爬措施

## 4. AI 工作流自动化

复杂任务可以拆分成工作流：
1. 数据收集
2. 数据处理
3. AI 分析
4. 结果输出

我把自己常用的功能开源了：
https://github.com/zhuxunyu/ai-productivity-toolkit

包含：
- Prompt 优化器
- Excel 批量处理
- 数据爬取框架
- AI 工作流

免费开源，欢迎 Star！

## 5. 其他推荐工具

- Notion AI - 笔记 + AI
- Zapier - 工作流自动化
- IFTTT - 简单自动化

---

**总结：** 工具是手段，效率是目的。选择最适合你的，而不是最多的。

如果觉得有帮助，欢迎点赞收藏~
```
