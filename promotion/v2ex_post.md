https://github.com/zhuxunyu/ai-productivity-toolkit

## 核心功能

### 1. Prompt 优化器
自动优化 AI 提示词，让 ChatGPT/Claude 更好地理解你的需求。

示例：
```python
from toolkit import PromptOptimizer
optimizer = PromptOptimizer()
optimized = optimizer.quick_optimize("帮我写个脚本")
print(optimized)
```

### 2. Excel 自动化工具
批量处理 Excel 文件，支持格式转换、数据合并等。

### 3. 数据爬取框架
合法合规的数据采集工具，支持批量抓取、数据导出。

### 4. AI 工作流
一键自动化复杂任务，如情感分析、内容生成等。

## 技术栈
- Python 3.8+
- openpyxl, pandas, requests, beautifulsoup4

## 使用方式
```bash
git clone https://github.com/zhuxunyu/ai-productivity-toolkit.git
cd ai-productivity-toolkit
pip install -r requirements.txt
python examples/prompt_optimizer_demo.py
```
