# 国产 AI 助手 Prompt 模板

针对主流国产 AI 助手优化的 Prompt 模板，提升回答质量。

---

## 📋 通用优化原则

1. **明确角色**: 告诉 AI 它应该扮演什么角色
2. **设定场景**: 说明使用场景和目标受众
3. **给出约束**: 明确字数、格式、风格要求
4. **提供示例**: 给出期望的输出样例

---

## 🎯 Kimi 专用 Prompt

### 长文本总结
```
你是一位专业的文献综述专家。请阅读我上传的论文，完成以下任务：

1. 用 300 字总结核心研究问题和方法
2. 列出 3-5 个关键创新点
3. 指出研究的局限性
4. 用通俗语言解释对普通人的意义

要求：
- 避免学术 jargon，用高中生能听懂的话
- 每个创新点用一句话概括
- 最后给出"值不值得读"的建议（1-5 星）
```

### 文件对比
```
我上传了 2 份文档，请对比它们的：
1. 核心观点差异
2. 论据强弱
3. 结论可信度

输出格式：对比表格 + 300 字分析
```

## 🎯 文心一言 专用 Prompt

### 中文写作优化
```
你是一位资深中文编辑。请帮我优化以下文字：

【待优化内容】
[粘贴你的文字]

优化方向：
1. 让表达更自然流畅，像真人说话
2. 删除冗余词汇
3. 增强情绪感染力
4. 保持原意不变

输出：优化后版本 + 修改说明（3-5 点）
```

### 多模态解读
```
请仔细分析这张图片：

1. 描述图片中的主要元素（客观）
2. 解读图片传达的信息/情绪（主观）
3. 如果是数据图表，指出关键趋势
4. 给出 3 个基于图片的洞察

要求：分点作答，每点 50-100 字
```

## 🎯 通义千问 专用 Prompt

### 代码生成
```
你是一位资深 Python 工程师。请完成以下任务：

【需求】
[描述你的需求]

要求：
1. 代码完整可运行
2. 添加详细注释（每段代码上方）
3. 包含错误处理
4. 提供使用示例
5. 说明时间/空间复杂度

输出格式：代码块 + 使用说明
```

### 技术方案设计
```
你是一位系统架构师。请为以下场景设计技术方案：

【场景】
[描述你的场景]

输出框架：
1. 技术选型（含对比）
2. 架构图（用文字描述）
3. 核心流程
4. 潜在风险及应对
5. 预估成本

要求：专业但易懂，避免过度设计
```

## 🎯 豆包 专用 Prompt

### 快速问答
```
请用 3 句话回答：[你的问题]

要求：
- 第 1 句：直接答案
- 第 2 句：关键原因/依据
- 第 3 句：延伸建议/注意事项
```

### 生活建议
```
你是一位生活达人。请给我建议：

【场景】[描述你的情况]

输出：
✅ 推荐做法（3 条）
❌ 避免做法（2 条）
💡 小技巧（1 个）

要求：实用、可操作、不说废话
```

## 🎯 智谱清言 专用 Prompt

### 论文解读
```
你是一位学术导师。请解读这篇论文：

1. 研究背景（为什么做这个研究）
2. 核心方法（用类比解释）
3. 实验结果（关键数据）
4. 学术贡献（对该领域的意义）
5. 延伸思考（后续研究方向）

要求：
- 专业术语需解释
- 关键公式用文字说明
- 给出"是否值得引用"的建议
```

### 学术写作
```
你是一位期刊审稿人。请帮我优化这段学术文字：

【待优化】
[粘贴文字]

优化方向：
1. 逻辑更严谨
2. 表达更精确
3. 符合学术规范
4. 增强说服力

输出：优化版本 + 修改理由（每处修改说明原因）
```

## 🔄 跨平台通用 Prompt

### 复杂任务拆解
```
请帮我完成以下复杂任务：

【任务】[描述任务]

请按以下步骤：
1. 先分析任务的关键难点
2. 拆解为 3-5 个子任务
3. 逐一完成每个子任务
4. 最后整合输出

要求：每步完成后询问我是否继续
```

### 批判性思考
```
请对以下观点进行批判性分析：

【观点】[粘贴观点]

分析框架：
1. 观点的合理之处（2-3 点）
2. 潜在的逻辑漏洞（2-3 点）
3. 被忽略的因素（1-2 点）
4. 更全面的表述建议

要求：客观中立，不预设立场
```

## 📊 各平台 Prompt 调优技巧

| 平台 | 优势 | Prompt 调优方向 |
|------|------|----------------|
| Kimi | 长文本 | 充分利用文件上传，要求结构化输出 |
| 文心一言 | 中文表达 | 强调"自然流畅"，要求口语化 |
| 通义千问 | 代码能力 | 明确要求注释、测试、复杂度分析 |
| 豆包 | 快速响应 | 限制字数，要求直接答案 |
| 智谱清言 | 学术场景 | 要求专业深度，允许术语 |

## 💡 高级技巧

### 1. 角色叠加
```
你同时是 [角色 A] + [角色 B]。请从两个视角分别分析...
```

### 2. 反向提问
```
在回答之前，请先问我 3 个澄清问题，确保你理解我的需求。
```

### 3. 输出约束
```
请用以下格式输出：
- 标题：不超过 20 字
- 正文：分 3 段，每段 100-150 字
- 总结：1 句话，不超过 30 字
```

### 4. 迭代优化
```
这是第一版：[粘贴内容]
请从以下角度优化：
1. 逻辑更清晰
2. 例子更具体
3. 语言更生动

输出优化版 + 修改说明
```
